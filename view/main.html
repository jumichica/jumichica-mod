Características

Sencillo

Solo debes generar el módulo con el código composer create-project jumichica/jumichica-mod [NOMBRE_MODULO].

Modelo de Datos

Gestión simplificada de datos y conexiones a la base de datos.

Controladores

Capa de control para el análisis o procesamiento de datos.

Ágil

Motor simplficado para la ejecución de la aplicación en PHP.

Seguro

Protocolo de protección de las interacciones y eventos.

Actualizado

Adaptado a las últimas versiones de PHP.

Estructura de Directorios
  • conf/ - variables de configuración del módulo.
  • controller/controller.php - procesos y cálculos.
  • model/Model.php - interacción de datos.
  • static/ - css, js e img.
  • view/ - Vistas y modales del módulo.
